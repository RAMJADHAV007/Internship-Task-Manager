# Task Management CLI Application

A simple and interactive Command-Line Interface (CLI) application built in Python to manage daily tasks. This project was developed as a final capstone project for the VaultofCodes internship program.

##  Goal
To build a practical, interactive application that helps users track, add, and complete daily tasks, solving the real-world problem of personal organization and productivity.

##  Tech Used
* **Language:** Python 3.x
* **Core Concepts:** Data Structures (Lists, Dictionaries), While Loops, Functions, and Error Handling.

##  Features
* **View Tasks:** Displays all current tasks with visual status indicators.
* **Add a Task:** Allows users to dynamically input and add new tasks to their list.
* **Mark as Completed:** Users can select specific tasks by their corresponding number to update their status.
* **Interactive Menu:** Continuous loop that allows multiple operations until the user chooses to exit safely.
